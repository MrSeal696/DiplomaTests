# Diploma Selenium Tests

Автоматизированные тесты пользовательского интерфейса с использованием Selenium WebDriver (C#).

## Описание

В проекте реализовано тестирование авторизации пользователя с использованием подхода Page Object Model (POM).

Тест выполняет:
- открытие страницы логина
- ввод логина и пароля
- нажатие кнопки входа
- проверку успешной авторизации

## Технологии

- C#
- Selenium WebDriver
- ChromeDriver

## Запуск

1. Открыть проект в Visual Studio  
2. Установить зависимости (Selenium.WebDriver, ChromeDriver)  
3. Запустить проект (Ctrl + F5)
